Esta pantalla es tu “inbox” de oportunidades. Solo muestra las que están en estado OPEN (abiertas). Cuando aceptás o ignorás una oportunidad, desaparece de acá porque deja de estar OPEN.

Qué tengo que ver: después de cada acción, arriba aparece un mensaje (verde=OK, rojo=error). Si generás DEMO correctamente, aparecen 3 tarjetas y el contador OPEN pasa a 3.

  • Generar DEMO: crea 3 oportunidades de prueba (para aprender). Esperá ver un mensaje verde y 3 tarjetas.
  • Generar: intenta crear oportunidades a partir de tu portafolio. Si tu portafolio está vacío, puede generar 0.
  • Evaluar con IA: si la IA está configurada, completa el bloque “IA” de cada tarjeta (score, acción, resumen).
  • Base de datos: muestra el historial completo (OPEN + cerradas). Útil para buscar y auditar decisiones.
  • Aceptar: cierra como ACCEPTED. Ignorar: cierra como IGNORED. En ambos casos, el badge rojo baja.

Si apretás un botón y “no pasa nada”, mirá arriba si apareció una caja verde/roja: eso confirma el resultado.
